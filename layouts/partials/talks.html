Tools to Inspire - Google has open sourced the tools we use to build coding tutorials for events small and large. You can use this tool to author and serve beautiful, mobile-friendly, interactive codelabs of your own, just like the ones you see at https://g.co/codelabs, Without having to write a single line of code!

Superpower User Manual - Did you know you have superpowers? The average person now has access to more data and more powerful tools and computing resources for analyzing that data than ever before. In this tutorial, we'll combine Google Cloud Platform services, along with modern programming tools to analyze and visualize some interesting and practical questions. You'll leave this session with a better sense of how much data analytic power is available right at your fingertips.

Democratizing AI - An overview of Google's AI APIs, which offer easy access to rich, pre-trained models via a single API call, as well as deploy your own TensorFlow or Keras models, and others options between those extremes (e.g. AutoML). This talk features lots of code and demos.

Maths Got Me Here - I get paid to solve puzzles, which is pretty much my dream job. Primarily intended for students and others interested in career planning, this talk summarizes some things I've learned along my 35 year journey in computing and software development.

Google Cloud Guided Tour - Cloud computing has gotten incredibly powerful but also very complex -- there's a lot to understand. In this session, I'll provide a guided tour of the Google Cloud Platform with an emphasis on practical examples and demos.
